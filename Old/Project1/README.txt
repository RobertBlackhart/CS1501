HugeInteger Class
Robert McDermot
January 2013

INTRODUCTION

This program implements several arithmetic algorithms, including addition, subtraction, naïve multiplication, and Karatsuba-Ofman multiplication for very large integers with thousands of decimal digits.

INSTALLATION

The following files should be present:

  HugeInteger.java
  TestHugeInteger.java

Compile and test with Java 5 or later:

  javac TestHugeInteger.java 
  java TestHugeInteger 

USAGE

The file TestHugeInteger.java demonstrates usage.  Note that
HugeInteger's are base 10.

PROBLEMS

No known problems.
